{multi-excerpt:name=privatenetwork.rename}
h3. {{privatenetwork.rename(oldName, newName)}}
Rename a private network, the original name is automatically created when a private network is needed

h4. {{Parameters}}
* oldName (string): Name of the public IP range, given by the cloud provider
* newName (string): New name of the public IP range

h4. {{Example}}
{noformat}
login = john
passwd = secret
category = privatenetwork
action = rename
space = John Doe
oldName = <string>
newName = <string>
{noformat}

h4. {{Return}}
True or Error

{multi-excerpt}

{multi-excerpt:name=privatenetwork.listFreeIP}
h3. {{privatenetwork.listFreeIP(name)}}
List of free IP addresses in a private network. A name of the private network must be provided with this function

h4. {{Parameters}}
* name (string): Name of a public IP range. The name is unique within space

h4. {{Example}}
{noformat}
login = john
passwd = secret
category = privatenetwork
action = listFreeIP
space = John Doe
name = <string>
{noformat}

h4. {{Return}}
[], list of free IP addresses

{multi-excerpt}

passwd = secret
category = privatenetwork
action = delete
space = John Doe
name = <string>

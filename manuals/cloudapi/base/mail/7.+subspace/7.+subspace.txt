{multi-excerpt:name=subspace.create}
h3. {{subspace.create(subSpace, processingCredits, storageCredits, networkingCredits, userFirstName, userLastName, userLogin, userPassword, email)}}
Create a new  sub-space

h4. {{Parameters}}
* subSpace (string): Name of the sub-space
* processingCredits (integer): Amount of processing credits, assigned to this space
* storageCredits (integer): Amount of storage credits, assigned to this space
* networkingCredits (integer): Amount of networking credits, assigned to this space
* userFirstName (string): First name of the user
* userLastName (string): Last name of the user
* userLogin (string): User login to access the new space
* userPassword (string): User password to access the new space
* email (string): E-mail address to reach the customer

h4. {{Example}}
{noformat}
login = john
passwd = secret
category = subspace
action = create
space = John Doe
subSpace = <string>
processingCredits = <integer>
storageCredits = <integer>
networkingCredits = <integer>
userFirstName = <string>
userLastName = <string>
userLogin = <string>
userPassword = <string>
email = <string>
{noformat}

h4. {{Return}}
True or Error

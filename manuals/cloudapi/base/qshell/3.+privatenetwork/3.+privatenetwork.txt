h2. {{PyMonkey Cloud API DOC for cloud object}} privatenetwork
{multi-excerpt:name=privatenetwork.list}
h3. {{privatenetwork.list()}}
List of all private networks

h4. {{Example}}
{code}
cloudApiConnection.privatenetwork.list()
{code}

h4. {{Return}}
array(dict)  [\{privateNetworkName:name, network:aipaddr, subnet:mask\}]

{multi-excerpt}

List of all private networks in a VPDC

h4. {{Parameters}}

h4. {{Example}}
{code}
cloudApiConnection.privatenetwork.listInVpdc(vpdc)
{code}

h4. {{Return}}
array(dict)  [\{privateNetworkName:name, network:aipaddr, subnet:mask\}]

{multi-excerpt}

{multi-excerpt:name=privatenetwork.rename}
h3. {{privatenetwork.rename(oldName, newName)}}
Rename a private network, the original name is automatically created when a private network is needed

h4. {{Parameters}}
* oldName (string): Name of the public IP range, given by the cloud provider
* newName (string): New name of the public IP range

h4. {{Example}}
{code}
cloudApiConnection.privatenetwork.rename(oldName, newName)
{code}

h4. {{Return}}
True or Error

{multi-excerpt}

{multi-excerpt:name=privatenetwork.listFreeIP}
h3. {{privatenetwork.listFreeIP(name)}}
List of free IP addresses in a private network. A name of the private network must be provided with this function

h4. {{Parameters}}
* name (string): Name of a public IP range. The name is unique within space

h4. {{Example}}
{code}
cloudApiConnection.privatenetwork.listFreeIP(name)
{code}

h4. {{Return}}
[], list of free IP addresses

{multi-excerpt}

{multi-excerpt:name=privatenetwork.create}
h3. {{privatenetwork.create(name)}}
Create a new private network. The cloud software will make sure a new VLAN (or comparable technology) is created with a unique name in the cloud domain.

h4. {{Parameters}}
* name (string): Name of a public IP range. The name is unique within space

h4. {{Example}}
{code}
cloudApiConnection.privatenetwork.create(name)
{code}

h4. {{Return}}
True or Error

{multi-excerpt}

{multi-excerpt:name=privatenetwork.getDetails}
h3. {{privatenetwork.getDetails(name)}}
Get the details of a private network

h4. {{Parameters}}
* name (string): Name of a public IP range. The name is unique within space

h4. {{Example}}
{code}
cloudApiConnection.privatenetwork.getDetails(name)
{code}

h4. {{Return}}
\{\}, \{name: string, customer: string, network: string, netmask: string,gateway: string, domainnameservers: string\}

{multi-excerpt}

{multi-excerpt:name=privatenetwork.delete}
h3. {{privatenetwork.delete(name)}}
Delete a private network if no machine is using it. An error is thrown if a machine uses the private network

h4. {{Parameters}}
* name (string): Name of a public IP range. The name is unique within space

h4. {{Example}}
{code}
cloudApiConnection.privatenetwork.delete(name)
{code}

h4. {{Return}}
True or Error

{multi-excerpt}
